# Versículo do dia

## Tecnologias utilizadas

React + TypeScript + Tawindcss

## Procedimentos
### Premissas
Ter o Node.js instalado

1. Forkar e clonar o repositório
2. Acessar o diretório clonado
3. Executar:
   ```
     npm install
   ```
   ```
     npm start
   ```
4. Aguarde, irá abrir o browser no endereço http://localhost:3000

Pronto!
